# E-Commerce Sales Analysis

## Problem Statement
Online store ka sales data analyze kiya taake pata kare
ki konsa product best sell hota hai, konsa region zyada
revenue deta hai, aur konse mahine mein sabse zyada
sales hoti hain.

## Tools Used
Python | Pandas | Matplotlib | Jupyter Notebook

## Dataset
Kaggle Sample Superstore — 9,994 rows, 21 columns

## Key Findings
- West region ne 32% zyada revenue generate kiya
- Technology category sabse zyada profitable rahi
- December mein sales 45% spike hoti hai

## Files
- `analysis.ipynb` — main Jupyter Notebook
- `data/superstore.csv` — raw dataset
- `charts/` — exported visualizations

## How to Run
1. Dataset download karo (link below)
2. Jupyter Notebook open karo
3. `analysis.ipynb` run karo cell by cell
